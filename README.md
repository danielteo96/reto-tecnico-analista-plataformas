Reto técnico para el puesto – Analista de Plataformas

Este proyecto corresponde al reto técnico solicitado para el puesto de Analista de Plataformas.  
Se trata de una API REST desarrollada en Node.js utilizando NestJS y MySQL.

con las tecnologías que se trabajo:
- Node.js
- NestJS
- TypeScript
- MySQL
- TypeORM

Se trabajo con la base de datos mysql que se compartio por correo:
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=********
DB_NAME=test_db

--Instalar 
npm install

npm run start:dev
